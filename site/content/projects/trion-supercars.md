## Overview

Full-stack design and development of a modern web presence for Trion Supercars, featuring interactive 3D vehicle configurator and company showcase.

## Design

- Designed complete user experience in Figma with stakeholder feedback
- High-fidelity mockups and component systems for consistency
- Mobile-first responsive approach across all breakpoints

## Development

**Frontend**
- Built with semantic HTML5, modern CSS3/SCSS with grid and flexbox
- Used AlpineJS for dynamic templating and component interactions
- Integrated Three.js for real-time 3D car model rendering and rotation
- Optimized WebGL performance for smooth 60fps interactions

**Features**
- Interactive car configurator (color, wheels, interior options)
- Smooth scrolling animations and transitions
- Performance-optimized 3D models with LOD support
- SEO-friendly markup and metadata

## Deployment

- Deployed to Netlify with automatic CI/CD from Git
- CDN optimization for global performance
- Analytics integration for user behavior tracking

## Impact

Website launched in 2025 and currently serves as the primary sales and marketing tool for Trion.

## Technologies

HTML5, CSS3, SCSS, JavaScript, AlpineJS, Three.js, Figma, Netlify
