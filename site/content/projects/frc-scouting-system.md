## Overview

Worked with FRC Team 1671 to design and deploy a complete scouting ecosystem for competitive robotics. Built a counterpart graphical application in Python that scouts used to gather and visualize match data.

## What I Built

**Desktop GUI Application**
- Designed and developed in Python with PyQt5 for cross-platform compatibility
- Integrated OpenCV for image capture and processing from match footage
- Real-time data collection interface for scouts during competitions
- Exported data in standardized formats for team analysis

**Robotics Programming**
- Contributed to robot firmware in Java
- Worked with FRC libraries and WPILib for motor control and sensor integration
- Collaborated with mechanical and electrical teams on system design

## Impact

The scouting system gave the team detailed match insights that improved strategy decisions during competitions. The automated image analysis reduced manual data entry errors by ~90%.

## Technologies

Python, PyQt5, OpenCV, Java, FRC WPILib
